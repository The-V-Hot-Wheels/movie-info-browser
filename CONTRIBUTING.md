# Guidelines for Hacktoberfest 2026 contributors

This project is open to Hacktoberfest 2026 contributors. Preferably Apple 
Developer Academy students.

## Requirements

* You must have Xcode.
* You must have [an OMDb API key](https://www.omdbapi.com/apikey.aspx). Place 
the key in secrets/APIKey.swift in "`let movieAPIKey = "555_EXAMPLE_5555"`", 
replacing "`555_EXAMPLE_5555"`" with your actual API key. However, before 
committing or even staging anything to Git, make sure that APIKey.swift is not 
being tracked by Git. Otherwise, the Git Ignore might need adjustment.
